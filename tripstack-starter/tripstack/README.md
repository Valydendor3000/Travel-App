# TripStack — Cloudflare + Expo white‑label travel app starter

This repository contains:
- **apps/mobile**: Expo React Native app (iOS/Android)
- **apps/admin**: Cloudflare Pages (Vite + React) admin dashboard
- **packages/api**: Cloudflare Workers API (with simple routes)
- **packages/db**: D1 schema/migrations
- **packages/ui**: shared components/styles
- **infra**: Cloudflare config + CI/CD

> You can upload this folder structure directly to GitHub. Fill `.env`/secrets in dashboards later.
